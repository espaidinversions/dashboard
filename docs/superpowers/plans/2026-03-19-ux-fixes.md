### 2b: Handle budget-only quarters in CompanyDetail

**File:** `src/components/CompanyDetail.jsx`

The chart already maps `quarters` → all shown. Budget-only quarters (no `rev`/`ebitda`/`dfn`) will render the budget bar but the actual bar will be 0/missing — that's acceptable. However:

- [ ] **Step 2: Ensure LTM skips future (budget-only) quarters**

  The `ltm` useMemo currently takes `quarters.slice(-4)`. This would include future budget-only quarters in the LTM sum, returning `0` for actuals. Fix: filter to quarters that have at least one actual value before slicing:

  ```jsx
  const ltm = useMemo(() => {
    if (quarters.length === 0) return null;
    const withActuals = quarters.filter(q => q.rev != null || q.ebitda != null || q.dfn != null);
    const last4 = withActuals.slice(-4);
    const sum = key => last4.reduce((s, q) => s + (q[key] ?? 0), 0);
    return { rev: sum("rev"), ebitda: sum("ebitda"), dfn: sum("dfn"), n: last4.length };
  }, [quarters]);
  ```

## Task 4: Fix J-curve — calls negative, distributions positive

**File:** `src/components/FundDetail.jsx`

**Root cause:** `cumCalls` accumulates positively. For a J-curve, capital called (outflows) must be negative so the net curve dips below zero before recovering.

- [ ] **Step 1: Flip cumCalls sign in jCurveData**

  In the `jCurveData` useMemo, change the accumulation and the returned value:

  ```jsx
  const jCurveData = useMemo(() => {
    const rows = txs
      .filter(r => r.cat === "Capital Call" || r.cat === "Distribució" || r.cat === "Retorn Capital")
      .sort((a, b) => a.data.localeCompare(b.data));
    let cumCalls = 0, cumDist = 0;
    return rows.map(r => {
      if (r.cat === "Capital Call") cumCalls += r.eur;
      else cumDist += Math.abs(r.eur);
      return { data: r.data, cumCalls: -cumCalls, cumDist };
    });
  }, [txs]);
  ```

  `cumCalls` is now always ≤ 0; `cumDist` is always ≥ 0.

- [ ] **Step 2: Add ReferenceLine at y=0 and import it**

  At the top of the file, `ReferenceLine` is not yet imported. Add it:
  ```jsx
  import {
    BarChart, Bar, XAxis, YAxis, Tooltip, ResponsiveContainer, CartesianGrid, ReferenceLine
  } from "recharts";
  ```

  Inside the `<BarChart>`, add after `<CartesianGrid .../>`:
  ```jsx
  <ReferenceLine y={0} stroke={tc.border} strokeWidth={1.5} />
  ```

- [ ] **Step 3: Update YAxis domain and tooltip formatter**

  The YAxis `domain` should include negatives. Let recharts auto-scale:
  ```jsx
  <YAxis tickFormatter={v => fmtM(Math.abs(v))} tick={{ fontSize: 10, fill: tc.textLight }} width={70} />
  ```

  Update the tooltip formatter to show absolute values with sign labels:
  ```jsx
  formatter={(v, name) => [
    (v < 0 ? "−" : "+") + fmtM(Math.abs(v)),
    name === "cumCalls" ? "Capital Cridat (acum.)" : "Distribucions (acum.)"
  ]}
  ```
